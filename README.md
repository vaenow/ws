##How to use
###### step 1: 
Deploy the [ws.mwb](https://github.com/vaenow/ws/tree/master/res/sql)(~/ws/res/sql/ws.mwb)
###### step 2: 
Run the command: 
```java
mvn jetty:run
```
###### step 3: 
Open `Google Chrome`, type 
```xml
http://localhost:8080
```
